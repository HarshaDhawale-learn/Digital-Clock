# ⏰ Digital Clock
A responsive digital clock built using HTML, CSS and JavaSript. This project features a modern UI with soft animated background elements for an aesthetic user experience.

## ✨ Features
-⏰ Live digital clock (updates every second)
-🌸 Cute animated background elements (flowers, bunnies, emojis)
-📱 Fully responsive (mobile-friendly)
-🎨 Custom pastel color theme

## 🔨 Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 What I Learned
- DOM Manipulation using JavaScript
- Working with 'setInterval()' for real-time updates
- CSS animation and keyframes
- Responsive design using media queries
  
## 🚀 How to Run
1. Clone the repository
2. Open 'digi1.html' in your browser
